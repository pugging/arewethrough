Privacy

Short version: we know nothing about you and we like it that way.

  • No accounts, no signup. The tool runs entirely in your browser.
  • Your scenarios are stored in the page URL only. If you share a link, whoever receives it sees the scores you typed — that's the feature.
  • We set no cookies.
  • Hosting: the site is served by GitHub Pages, which may log standard technical request data (IP address, user agent) per GitHub's privacy statement.
  • Analytics: if we enable privacy-friendly counting (GoatCounter) or ads (Google AdSense) later, this page will say so and link the relevant policies.
  • Affiliate links: if we add them, they'll be marked and we earn a commission at no cost to you.
