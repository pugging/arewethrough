# Value Chain (цепочка ценности по Портеру) — AreWeThrough?

*Совет директоров Milkabets · 13 июня 2026*

Цепочка для цифрового продукта: от сырых данных до удержанного болельщика. Маржа создаётся на каждом звене за счёт нулевой себестоимости и автоматизации.

## Основные активности (primary)

| Звено | Что происходит | Наше преимущество |
|---|---|---|
| **Inbound logistics** — данные | Парсер тянет 12 страниц Википедии → валидация (72 матча/48 команд, ассерты) → data.js | Автоматизация: 1 команда, 30 секунд; ассерты ловят битые данные до публикации |
| **Operations** — расчёт | Движок в браузере: таблицы → тай-брейки ФИФА → рейтинг третьих → вердикт | Точность правил (личные встречи, ⚖-пометки) = доверие; всё на клиенте = ноль серверных затрат при любом трафике |
| **Outbound logistics** — доставка | GitHub Pages CDN, статика, ~30 КБ JS без зависимостей | Выдерживает reddit hug of death бесплатно |
| **Marketing & Sales** | Шар-ссылки (виральная петля), 48 SEO-страниц, ответы в матч-тредах, OG-карточки в мессенджерах | Каждый спор в чате = бесплатная дистрибуция; конкуренты тратят это место на баннеры |
| **Service** | «How it works» снимает споры о правилах; Privacy для AdSense; issues на GitHub | Самообслуживание, поддержка ≈ 0 |

## Поддерживающие активности (support)

- **Инфраструктура:** GitHub (репо+хостинг+CI потенциально), бюджет $0.
- **HR:** 2 фаундера + 1 робот. Опс-нагрузка ~20 мин/день (обновить счёт, ответить в тредах).
- **Технологии:** vanilla JS (нет цепочки зависимостей = нет поломок в турнирные дни), юнит-тесты движка, генератор страниц.
- **Закупки:** нет. Опционально: домен $12/год, когда докажем трафик.

## Где создаётся и захватывается ценность
1. **Создание:** превращаем открытые данные (бесплатные) в ответ на вопрос, который болельщик не может посчитать сам, — в момент максимальной эмоции.
2. **Захват:** внимание на пике → партнёрский интент (VPN «как смотреть», мерч «новая любимая сборная») + рекламный RPM + актив (бренд, обратные ссылки, email-лист) для следующего турнира.
3. **Защита:** сетевой эффект шар-ссылок + длиннохвостые страницы, которые сквоттеры не успели занять под конкретные команды.
