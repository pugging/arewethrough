# Business Model Canvas — AreWeThrough?

*Совет директоров Milkabets · 13 июня 2026 · v1.0*

## 1. Customer Segments (сегменты)
- **Ядро:** болельщики 48 сборных на ЧМ-2026, следящие за групповым этапом (пик: 22–27 июня). Объём: сотни миллионов зрителей, из них десятки миллионов активно гуглят расклады.
- **Суперядро:** болельщики команд, идущих 3-ми в группе, — единственные, кому нужна таблица третьих мест (наша уникальная боль).
- **Вторичный:** спорщики в групповых чатах; журналисты/блогеры, которым нужна интерактивная ссылка вместо статичной таблицы.

## 2. Value Propositions (ценность)
- «Введи счёт — мгновенно увидь, выходит ли твоя сборная» — расчёт, который в формате 2026 года (12 групп + 8 лучших третьих) невозможно сделать в голове.
- **Шар-ссылка сценария** — раскладом можно доказать правоту в чате (уникально среди конкурентов).
- Без регистрации, без приложения, бесплатно, мгновенно, работает на телефоне.
- Точные правила ФИФА (включая личные встречи и пометку «жребий») — доверие.

## 3. Channels (каналы)
- Reddit (r/worldcup, r/soccer, страновые сабы) — ответы на реальные вопросы «what do we need».
- Виральная петля шар-ссылок в мессенджерах (WhatsApp/Telegram/iMessage).
- SEO-долгий хвост: 48 страниц «What does X need to qualify» + страница правил.
- X/Twitter: ответы на твиты журналистов о пермутациях.

## 4. Customer Relationships (отношения)
- Self-service, нулевая поддержка. Доверие через точность и прозрачность правил («How it works»).
- Возврат: закладка страницы своей команды до последнего тура.

## 5. Revenue Streams (доходы)
- **Сейчас:** донаты (Ko-fi) — копейки, символически.
- **Дни 2–5:** партнёрки — VPN ($15–40/подписка, интент «смотреть трансляцию из-за границы»), фан-атрибутика Fanatics/Kitbag (5–10% с заказа).
- **Неделя 2:** AdSense на пике трафика ($3–15 RPM, US-трафик дорогой).
- **После турнира:** движок перекрашивается под ЛЧ/Евро-2028/Копа — сезонная франшиза + email-список.

## 6. Key Resources (ресурсы)
- Движок расчёта (протестирован, 15 юнит-тестов) + данные матчей (автопарсер Википедии).
- Бренд/домен arewethrough + контент 50 страниц.
- Ноль инфраструктурных затрат (статика на GitHub Pages).

## 7. Key Activities (активности)
- Ежедневное обновление результатов (1 команда: `python3 tools/update_data.py && node tools/build.js && git push`).
- Комьюнити-дистрибуция в матчдни; ответы на вопросы ссылками-сценариями.
- Подача в партнёрки/AdSense/Search Console (день 1–2).

## 8. Key Partnerships (партнёры)
- GitHub Pages (хостинг), Wikipedia (данные, CC BY-SA — указываем источник).
- Партнёрские сети: Impact/CJ (VPN, Fanatics), Google AdSense.
- Неформально: футбольные сабреддиты и блогеры (даём им embed/ссылки).

## 9. Cost Structure (затраты)
- Деньги: **$0** обязательных (опционально домен ~$12/год).
- Время: 1 ночь сборки + ~20 мин/день операций до 27 июня.
- Риски-затраты: ручная проверка результатов в дни матчей (репутация = точность).

**Юнит-экономика спайка (модель):** 100k просмотров пиковой недели × ($5 RPM) ≈ $500 + 50 VPN-конверсий × $25 ≈ $1 250 + мерч ≈ $100–300. Затраты $0. LTV актива: переиспользование на 3+ турнирах/год.
